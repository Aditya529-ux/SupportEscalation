# 🚀 SupportEscalation

An AI-powered customer support system built using **Python, Streamlit, LangChain, LangGraph, SQLite, and ChromaDB**.

---

## 📖 Project Overview

SupportEscalation is a multi-agent customer support application that intelligently routes user queries to specialized AI agents.

Instead of using one AI model for everything, this system first understands the user's request and then forwards it to the appropriate support agent.

Example:

User
↓
Router Agent
↓
Billing Agent / Technical Agent / Account Agent / General Agent
↓
Response
↓
Escalation (if required)
